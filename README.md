# PHP Stock Tracking System

Product and Stock Management System

# Techstack

- PHP
- Bootstrap
- jQuery
- Html-CSS
- JavaScript
 
 # Features
 
- User Login Panel
- Stock Dashboard 
- Stock Adding
- Stock Manage / Edit / Delete
- Settings for brand and category
- Product Category Manage
- Orders Management
- Print Order
- Preparing Order Report
- User Settings(Change Username/Password)
